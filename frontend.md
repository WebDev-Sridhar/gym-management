# 🎨 FRONTEND DESIGN SYSTEM (Gym SaaS)

## 🌑 LANDING PAGE THEME

* Dark theme only
* Background: near black (#0B0B0F)
* Glassmorphism + subtle gradients
* Neon accent colors (purple / blue glow)
* High contrast typography

## ✨ VISUAL STYLE

* Premium SaaS (not template-like)
* Inspired by:

  * modern startup websites
  * futuristic dashboards
* Avoid:

  * generic cards
  * boxy layouts

## 🔤 TYPOGRAPHY

* Headings: Bold, large, tight spacing
* Body: clean, readable
* Use modern fonts (Inter / Geist / Satoshi style)

## 🎭 ANIMATIONS (LANDING PAGE)

* Smooth page transitions
* Scroll-based reveals
* Hover interactions:

  * lift
  * glow
  * scale
* Micro-interactions everywhere

## 🎬 ANIMATION LIBRARY

* Use Framer Motion

## 🎯 INTERACTIONS

* Buttons:

  * hover glow
  * slight scale
* Cards:

  * hover elevation
* Sections:

  * parallax or fade-in

## 🚫 AVOID

* Static layouts
* Bootstrap-style UI
* Overcrowded sections

---

## 🌕 DASHBOARD & MEMBER APP

* Default: Light theme
* Clean, minimal UI
* Focus on usability, not heavy animation

## ⚡ DASHBOARD ANIMATION

* Subtle only:

  * hover
  * transitions
* No heavy motion

## 📱 MEMBER APP

* Mobile-first
* Fast, simple
* No distractions

---

## 🎯 DESIGN PRINCIPLES

* Clarity > decoration
* Motion with purpose
* Every animation must guide user attention
* Premium feel over flashy chaos
